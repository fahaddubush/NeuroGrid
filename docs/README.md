# NeuroGrid documentation

Documentation is organized by purpose:

| Directory | Contents |
| --- | --- |
| [`architecture/`](architecture/) | System topology, agent design, tier responsibilities, and operational flow |
| [`guides/`](guides/) | Implementation and operating guides for major execution paths |
| [`research/`](research/) | Research motivation, methodology, and the project paper |

The root [README](../README.md) provides the supported setup and validation workflow.
[SECURITY.md](../SECURITY.md) defines the current security boundaries. Source code and tests
remain authoritative when older research descriptions differ from the implementation.
